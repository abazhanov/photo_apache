
1) header.php - Шаблон шапки страницы. Там лежит:
    - Яндекс метрика
    - Меню
2) footer.php - Шаблон подвала страницы. Там лежит:
    - Футер с иконками соц.сетей

3) contacts.php - Контактные данные
4) copyright.php - мой копирайт. При приобретении неисключительных прав, этот файл можно удалить, но при этом, необходимо удалить из файла Template.cshtml следующую строку: <div class="copyright"><a href="copyright.cshtml">COPYRIGHT © Artem Bazhanov</a></div>
4) galleries.php - страница с сылками на фотогалереи.
    - Добавление фотогалереи:
        а) В первую очередь, необходимо создать новый файл с расширением .php, содержащий код:
            <?php include ('header.php'); ?>
            <?php include ('footer.php'); ?>
        б) Между этих строк, необходимо вставить код, сгенерированный для нашей новой галерии на ресурсе http://cincopa.com
        в) Далее, необходимо добавить в список строк аналогичную строку по шаблону:
            <li><a href="ИМЯ_ФАЙЛА.php"><img src="img/photo/galleries/ИМЯ_ГАЛЕРИИ/title-ИМЯ_ФАЙЛА.jpg" alt="ОПИСАНИЕ ГАЛЕРИИ"></a><div class="item">ОПИСАНИЕ ГАЛЕРИИ</div></li>
            
            пример из файла galleries.php:
            <li><a href="interior.php"><img src="img/photo/galleries/interior/title-interior.jpg" alt="Интерьер"></a><div class="item">Интерьер</div></li>
        г) Для отображения фото-миниатюры галерии, необходимо создать папку с именем галерии (желательно созвучно с именем создаваемой галерии) по адресу: /img/photo/galleries/ИМЯ_ГАЛЕРИИ и положиь туда файл с фото-миниатюрой, который должен носить имя из пункта 4-в: title-ИМЯ_ФАЙЛА.jpg с размером 300х200 пикселей.
    - удаление фотогалерии:
        а) удалить из списка соответсвующую строку:
            <li><a href="ИМЯ_ФАЙЛА.php"><img src="img/photo/galleries/ИМЯ_ГАЛЕРИИ/title-ИМЯ_ФАЙЛА.jpg" alt="ОПИСАНИЕ ГАЛЕРИИ"></a><div class="item">ОПИСАНИЕ ГАЛЕРИИ</div></li>
        б) при необходимости, удалить соответствующую папку из /img/photo/galleries
            
5) index.php - фоновая картинка главной страницы и текстовый блок с приветствием.
6) services.php - описание услуг
7) файлы: interior.php, wedding.php, lovestory.php, studio.php - являются созданными файлами фото галереи из пункта 4.
